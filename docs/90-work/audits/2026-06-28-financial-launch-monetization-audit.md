## TL;DR

- **Щоб _запустити_** і тримати продукт у повітрі треба дуже мало: **~₴1 200–3 200/міс** інфраструктури + **~₴3 000–12 000** одноразово (ФОП, домен, опц. Apple/Google акаунти). Майже весь стек уже на безкоштовних тірах (Vercel, Sentry, PostHog, Grafana, Resend).
- **Головний змінний кост — не хостинг, а Anthropic Claude API**, і він масштабується від **усіх** активних юзерів (free-тір дає 15 AI-повідомлень/день), а не лише від платників.
- **Беззбитковість при активній ціні ₴199/міс ≈ 18–22 Pro-підписники** (а не 30–35, як досі написано в канонічних доках — вони рахують на стару ціну ₴99). Це **головна знахідка аудиту**.
- ~20 Pro → це **≈400–670 MAU** при реалістичній freemium-конверсії 3–5 %.
- **Вихід у відчутний прибуток:** ~100 Pro ≈ **₴8–9 тис чистими/міс**; ~250 Pro ≈ **₴21 тис/міс**; year-1 ціль ₴100K MRR (~620 Pro) ≈ **₴55–60 тис/міс** до маркетингу й податків.

> **Дві різні бази (важливо для §4).** «ФІКС-інфра» — це те, що ти платиш, навіть з нуля юзерів (хостинг, email, observability): на launch-стадії лише **~$18–25/міс**. «Операційні» додають зверху змінний AI-кост, який росте від кількості активних юзерів (free-тір теж споживає AI).

**Планувальний baseline для §4 breakeven:** **~~$56–72/міс ≈ ₴2 460–3 170/міс** = ФІКС-інфра (~$20) **+** змінний Anthropic-на-free-юзерів (~~$30–50 при кількох сотнях MAU на breakeven-масштабі). Я свідомо включаю AI-на-free у baseline (а не лише чистий фікс ~$20), бо на breakeven-точці вже є сотні free-юзерів, що генерують реальний AI-рахунок — це консервативніше й не недооцінює поріг. Чистий-фікс-only breakeven був би ще нижчий (~10–14 Pro).

> **Знахідка F-2 (P1):** **Anthropic — це COGS на _всіх_ юзерів, а не лише на Pro.** Free-тір дає 15 AI-повідомлень/день (ADR-0068). При 10K MAU неоплачуваний AI-трафік може домінувати над усім іншим костом. Захист уже в коді: денний soft/hard cap (`ANTHROPIC_BUDGET_SOFT_USD=3` / `_HARD_USD=5`), tiered-degradation, prompt caching (−90 % на cached input), Haiku для first-turn. Це треба тримати ввімкненим **з дня 1** — інакше unit-економіка пливе.

> **Знахідка F-3 (P2):** **Reverse trial = AI-кост на кожного нового юзера 7 днів.** Кожен реєстрант отримує повний Pro (безлімітний AI) на 7 днів без картки. Це конверсійно сильно, але створює реальну змінну статтю: `нові_реєстрації × 7днів × Pro-AI-кост`. Має бути в моделі й під cost-guard (інакше сплеск реєстрацій = сплеск рахунку Anthropic).

## 3. Активна модель монетизації (ADR-0068)

| План     | Ціна                                        | Free-ліміти                                          | Pro-розблокування                                                                                       |
| -------- | ------------------------------------------- | ---------------------------------------------------- | ------------------------------------------------------------------------------------------------------- |
| **Free** | ₴0                                          | 15 AI-повідомлень/день, sync 2 пристрої, Mono ручний | —                                                                                                       |
| **Pro**  | **₴199/міс** або **₴1 490/рік** (~₴124/міс) | —                                                    | Безлім AI, авто-Mono sync (+backfill, мульти-банк), безлім пристроїв, експорт, weekly digest, AI memory |

- **Trial:** 7 днів reverse trial (старт у Pro → авто-downgrade, без картки).
- **Провайдер:** Stripe інтегровано в коді (`/api/billing/checkout|status|portal|stripe-webhook`); для UA-старту рекомендований LiqPay/WayForPay (нижча комісія, рідний checkout) — `01 §4`.

> **Знахідка F-4 (P2): Stripe-комісія непропорційно б'є по дешевій місячній ціні.** Stripe = 2.9 % + **30¢ фіксовано**. На ₴199 це ≈ ₴5.8 + ₴13.2 = **~₴19/міс (≈9.5 %)**. LiqPay 2.75 % без фікс-фі = **~₴5.5/міс (2.75 %)**. **Річний план амортизує фікс-фі** (одна транзакція/рік замість 12) → ще один аргумент агресивно штовхати ₴1 490/рік. Рекомендація: LiqPay/WayForPay як основний UA-провайдер, Stripe — для міжнародних карток.

## 4. Беззбитковість (перерахунок на активну ціну ₴199)

### 4.1 Юніт-економіка одного Pro

```
Ціна (monthly Pro)            = ₴199
Ціна (annual Pro, на місяць)  = ₴1490 / 12 = ₴124
Blended ARPU (50/50)          = ₴161.5/міс

Змінний кост на 1 Pro/міс:
  Платіжна комісія    ≈ ₴6 (LiqPay/annual) … ₴19 (Stripe monthly) → беру ₴10
  Anthropic marginal  ≈ ₴5–15 (безлім AI під cost-guard)          → беру ₴10
  ──────────────────────────────────────────────
  Разом variable      ≈ ₴20/міс на Pro

Внесок (contribution) на 1 Pro:
  monthly:  ₴199  − ₴20 = ₴179
  blended:  ₴161.5 − ₴20 = ₴141.5
```

### 4.2 Точка беззбитковості

```
Фіксовані витрати (launch baseline)   = ₴3 170/міс ($72)

Breakeven Pro = Fixed / contribution
  при all-monthly (₴179):   3 170 / 179 ≈ 18 Pro
  при blended    (₴141.5):  3 170 / 141.5 ≈ 22 Pro
  при leaner fixed ₴2 460:  14–18 Pro

→ БЕЗЗБИТКОВІСТЬ ≈ 18–22 Pro-підписники (округлено ~20).
```

> **🔴 Знахідка F-1 (P1) — головна.** Канонічні доки рахують беззбитковість на **застарілій** ціні ₴99 і дають **35 Pro** (`01-monetization-and-pricing.md §9.4`) та **~30 Pro** (`03-services-and-toolstack.md §6`). Активна ціна за ADR-0068 — **₴199**, тому реальний breakeven **~18–22 Pro**, тобто вдвічі легший. Розрахунки в §9.4 і §6 треба синхронізувати з ADR-0068 (окремий PR; зачіпає канонічні доки → Rule #26 pr-ledger).

### 4.3 Скільки треба MAU, щоб зібрати ~20 Pro

Freemium-конверсія B2C productivity: медіана **3–5 %**, топ-перформери **8–12 %** (`01 §1`).

| Конверсія free→Pro | MAU для беззбитковості (~20 Pro) |
| ------------------ | -------------------------------- |
| 3 % (консерв.)     | **≈ 670 MAU**                    |
| 5 % (реалістично)  | **≈ 400 MAU**                    |
| 8 % (топ)          | **≈ 250 MAU**                    |

> Тобто щоб «вийти в нуль», достатньо **кількох сотень активних юзерів**. Це досяжно вже на closed-beta + Product Hunt хвилі (ціль `02-go-to-market`: 1K–5K signups першого тижня).

## 5. Сценарії виходу в прибуток

Костова база росте з MAU (переважно Anthropic). Моделюю 4 щаблі; ARPU blended ₴161.5, variable ₴20/Pro.

| Щабель         | Pro  | ~MAU (3–4 %) | MRR    | Variable | Fixed (з ростом AI) | **Чистий прибуток/міс** | ~$/міс |
| -------------- | ---- | ------------ | ------ | -------- | ------------------- | ----------------------- | ------ |
| Беззбитковість | ~20  | ~500–670     | ₴3.2K  | ₴0.4K    | ₴3.2K               | **≈ ₴0**                | $0     |
| Ранній profit  | 100  | ~2.5–3.3K    | ₴16.2K | ₴2.0K    | ₴5.5K               | **≈ ₴8.7K**             | ~$197  |
| Зростання      | 250  | ~6–8K        | ₴40.4K | ₴5.0K    | ₴14K                | **≈ ₴21.4K**            | ~$486  |
| Year-1 ціль    | ~620 | ~15K         | ₴100K  | ₴12.4K   | ₴30K                | **≈ ₴57.6K**            | ~$1.3K |

> **Застереження (P1):** числа вище — **до** маркетингу й податків.
>
> - **Реклама/ acquisition:** `03 §6` закладає 5–15K UAH/міс (growth) → 15–50K UAH/міс (scale). На щаблях 250+ Pro це може зʼїсти більшу частину «чистого» прибутку, якщо CAC не тримати низьким (ціль blended CAC ₴20–40, LTV:CAC 20–80:1 — `01 §9`).
> - **Податки ФОП 3-ї групи:** 5 % з обороту + ЄСВ (~₴1 760/міс). На ₴100K MRR це ≈ ₴5K податку + ЄСВ ≈ **₴6.8K/міс**.
> - **Зарплата/дровка засновника** не закладена взагалі — це bootstrap-модель «продукт сам себе утримує», не «продукт годує команду». Для FT-доходу засновника (~₴40–60K/міс) орієнтир — **щабель 250–400 Pro**.

## 8. Найбільші ризики моделі
