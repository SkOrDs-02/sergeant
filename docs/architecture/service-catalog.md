# Service Catalog

> **Last validated:** 2026-05-04 by @Skords-01. **Next review:** 2026-08-02.
> **Status:** Active

Runtime-oriented inventory for the production surfaces that Sergeant operates today. Use this document to answer four questions fast:

1. What service or surface is affected?
2. Where is it deployed?
3. What does it depend on?
4. How do we verify or roll it back?

## Operating rules

- Every release-affecting PR should name the touched surface from this catalog.
- If a surface changes deploy target, owner, healthcheck, or rollback path, update this catalog in the same PR.
- If a new production surface appears, add it here before calling the rollout complete.
