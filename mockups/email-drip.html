Notes · як вибирати

  • V1 Editorial — для тих хто хоче personal letter feel. Lora serif subject читається як друг написав. Підходить broad UA mass-market, особливо Routine/Nutrition audience.
  • V2 Branded — найbalanced. Module color cue (для Day 5 conditionally branded per module). Емodzi кнопки опційно. Default рекомендую — найкраще recognizable у inbox між іншою кореспонденцією.
  • V3 Mono — niche, для UA tech audience (DOU.ua subscribers, X UA-dev folks). Сильно diff від решти inbox. Ризик: відштовхне non-tech.
  • Усі 3 варіанти — однаковий copy backbone. Різниця лише у візуальному обрамленні. Можна A/B test між V2 (default) і V1 (alternative) у Phase 4.
  • Per-module Day 5 opener bank — у doc'у [`launch-plan.md`](../docs/marketing/launch-plan.md). Тут show'нуто Finyk варіант; n8n branchить per module_interest.
