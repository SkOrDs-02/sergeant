Notes · як вибирати

  • Subtitle 30 chars limit (iOS): "Усе про себе — в одному місці" = 30 symbols. Tight, але fits. Альтернатива "4 модулі · 1 застосунок" = 23.
  • V1 — рекомендую для App Store. iOS searchers скролять screenshots первым.
  • V2 — рекомендую для Play Store. Android users більше читають описи.
  • V3 — найкраще для screenshot 1 (cover screen). Module grid одразу пояснює value prop ще до того як user тапне.
  • Гібридна рекомендація: V1 layout + V3 module grid як screenshot 1 + V2 benefit list як screenshot 5 (last).
