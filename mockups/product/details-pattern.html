3 екрани на модуль — один патерн для всіх чотирьох

Цей файл показує зразок патерну на прикладі Фініка: огляд → деталь → створення. Інші три модулі (Фізрук, Рутина, Харчування) повторюють ту саму структуру з модульним колірним акцентом і відмінним змістом. Замість 12 окремих мокапів — один канонічний приклад + таблиця вмісту для решти.

Скоп цього мокапу

Замість малювати 4 модулі × 3 екрани = 12 артбордів, цей файл задає патерн один раз і пропонує його реплікацію для інших модулів коли вони знадобляться у production. Огляд (Overview) для Фінік + Фізрук + Рутина + Нутриція вже існує у product/<module>/. Тут — деталь і створення для Фініка як зразок, плюс вміст-таблиця що має бути в решти.

Що має бути в інших модулях

3 × 3 = 9 екранів

Той самий 3-екранний патерн (огляд → деталь → створення) застосовується до Фізрука, Рутини, Харчування. Колірний акцент змінюється, зміст модульний. Таблиця нижче — що саме у кожному.

Фінік

Гроші

  • огляд — транзакції за тиждень · hero ₴ + категорії
  • деталь — одна транзакція · сума, картка, категорія, нотатка
  • створення — нова витрата · сума + AI-категорія
Фізрук

Тіло

  • огляд — тренування за тиждень · hero хв + графік сили
  • деталь — одне тренування · сети, ваги, нотатки
  • створення — почати тренування · план + AI-recom вправ
Рутина

Звички

  • огляд — тиждень звичок · hero стрік + календар heatmap
  • деталь — одна звичка · історія, тригери, контекст
  • створення — нова звичка · ціль + час + дні тижня
Харчування

Їжа

  • огляд — прийоми за день · hero ккал + macro ring
  • деталь — один прийом · страви, інгредієнти, ккал
  • створення — нова страва · сканер / фото / пошук / ручне
