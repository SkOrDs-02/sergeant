V3Marketing usage

Do/Don't для marketing assets. Voice + visual rules з прикладами. Targeted на content production.

VOICE · DO
«Знаєш, куди йдуть гроші — без таблиць.»
«Один тап на день. Через місяць — побачиш картину.»
VOICE · DON'T
«Контролюй свої фінанси!»
«Ти вже Х днів не тренувався 😢»
«Нарешті — рішення для всіх твоїх задач»
COLOR · DO
Module color для CTA · text-strong tier на body sizes · Hub hero = soft & organic · module hero = bright tinted
COLOR · DON'T
Module color поза module surface · raw `bg-emerald-500` без `-strong` варіанта на body text · Cross-module palette mixing
